Welcome to Salt Framework

This is a simple web framework built with Python.

Routes

  • /home - Returns a "hello" message.
  • /about - Returns an "about" message.
  • /hello/{name} - Returns a personalized greeting.
  • /sum/{num_1}/{num_2} - Returns the sum of two numbers.
  • /book - Returns a "Books Page" message for GET requests, and a "Endpoint to create a book" message for POST requests.
  • /sample - Returns a "sample" message.
  • /template - Renders a template with the name "Template Name" and the framework name "Salt Framework".
